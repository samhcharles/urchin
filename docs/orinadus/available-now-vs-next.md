# Orinadus Capability Table: Available Now vs Planned Next

Use this table in marketing pages, docs, and onboarding to prevent promise drift.

## Summary

- Available now: Urchin capabilities that exist and are usable today.
- Planned next: Orinadus hosted product capabilities in staged delivery.

## Capability Matrix

| Area | Available now (Urchin) | Planned next (Orinadus) |
|---|---|---|
| Multi-tool capture | Yes. Captures activity from supported local sources and intake channels. | Expanded connectors and managed setup flows. |
| Unified memory archive | Yes. Activity can be synced into an Obsidian-based archive flow. | Hosted, managed memory views and guided organization. |
| Querying context | Yes. MCP tools and local workflows support retrieval of recent/project/topic context. | Dashboard search and natural-language interaction in product UI. |
| Brain hatch / onboarding | Limited. Existing users can wire Urchin into a current vault or scaffold a starter vault locally. | Guided guest flow, sign-in path, and export-friendly onboarding through Orinadus. |
| Cross-project organization | Partial. Project-context retrieval exists via current Urchin workflows. | Full dashboard-level project and brain management UX. |
| Visual relationship graph | No. | Planned: in-house visual graph for linked brain context. |
| Chat assistant over memory | No dedicated Orinadus UI yet. | Planned: Overseer assistant chat experience. |
| Team controls and collaboration | Limited in current open-source flow. | Planned: team-oriented hosted controls and collaboration features. |
| Guided onboarding | CLI/docs-first today. | Planned: opinionated product onboarding and setup assistant. |
| Multilingual support in UI | Not a product-layer feature today. | Planned: product UI and assistant language support. |

## Claim Safety Rules

1. If a feature is not generally usable today, label it planned.
2. Do not describe planned capabilities with present-tense product language.
3. Keep one sentence that clarifies: "Urchin is open source now; Orinadus hosted features are rolling out in phases."
4. Treat dashboard, graph, and Overseer chat as roadmap until shipped.

## Suggested Public Copy

Urchin is available now as the open-source memory substrate.
Orinadus is the hosted product layer that is rolling out in phases, starting with onboarding and access.

## Release Gates for Moving Items to "Available now"

A planned item can move to available only when all are true:

1. The feature is usable by external users.
2. Setup instructions exist and are tested.
3. One short demo path can be completed end-to-end.
4. Support and issue routing are defined.
